# Projeto: REST API de Usuários

## 📄 Descrição

Este projeto é uma aplicação simples que implementa uma **REST API** utilizando **Spring Boot**, com uma interface em HTML para visualização dos usuários cadastrados em um banco de dados.

A aplicação permite:
(CRUD) - completo

- Exibir **todos os usuários** cadastrados;
- Buscar e exibir um **usuário específico** a partir do **ID**
- Cadastrar novos usuários
- Editar usuários
- Deletar usuários

---

## Tecnologias Utilizadas

- **Java 21**
- **Spring Boot 3.5.3**
- **Spring Web**
- **Spring Data JPA**
- **Banco de Dados MySQL**
- **HTML + CSS + JavaScript**

---

## EndPoints
- localhost:8080/user (POST)
- localhost:8080/user/listaUsuarios (GET)
- localhost:8080/user/{id} (GET)
- localhost:8080/user/{id} (PUT)
- localhost:8080/user/{id} (DELETE)
